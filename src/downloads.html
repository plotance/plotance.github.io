Downloads

If you are not sure which to download, choose “Windows (Intel/AMD processors)”.

  • Windows (Intel/AMD processors)
  • Windows (Snapdragon and other ARM processors)
  • macOS
  • Linux (x86-64)
  • Linux (AArch64)

After downloading, extract the archive and run the plotance executable in the command prompt or terminal.

Windows:

.\plotance --help

Linux or macOS:

./plotance --help

For more information, see the documentation.
